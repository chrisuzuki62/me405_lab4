def main.print_data ( )

Function for printing collected data.

This function prints the each of the collected data as well as a value which for the time. Because the timer callback is run at 1000Hz, it is assumed that each value in the queue corresponds with a delta time of 1 millisecond. There is a marker that gets printed at the very end which indicates that all the data has been printed. This is important because on the PC side, if the readline() function is run with nothing in the serial port, it blocks the whole script from running.

◆ ticker()

def main.ticker (   timerobj)

Timer callback to collect the data from the step response.

This is a callback function for the timer which reads from the ADC and places the value in a queue. If the queue is full, no data is put in the queue to prevent our script from blocking.
